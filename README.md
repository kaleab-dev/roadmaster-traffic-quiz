RoadMaster is an interactive quiz website designed to help learners test their knowledge of traffic signs and road rules.  
Built using HTML, CSS, and JavaScript.

## Features
- Multiple-choice quizzes
- Score tracking
- Interactive interface
- Helps users prepare for driving theory tests

## Inspiration
This project was inspired by my interest in road safety and my personal experience with driving education. It also connects to my journey of overcoming fear related to driving.

## How to Use
1. Open `index.html` in your browser.
2. Click the "Get Started" button to begin.
3. Answer questions and see your score at the end.
4. Click "Finish Attempt" to finish the quiz.
